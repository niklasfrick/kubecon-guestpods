**Step 3: Create `server/store.go`**
SQLite data access layer:
- `NewStore(dbPath string) (*Store, error)` -- opens SQLite with WAL mode, busy_timeout=5000, synchronous=NORMAL, foreign_keys=ON, MaxOpenConns=1. Creates table if not exists:
  ```sql
  CREATE TABLE IF NOT EXISTS submissions (
      id            INTEGER PRIMARY KEY AUTOINCREMENT,
      name          TEXT    NOT NULL,
      country_code  TEXT    NOT NULL,
      homelab_level INTEGER NOT NULL CHECK(homelab_level BETWEEN 1 AND 5),
      created_at    DATETIME DEFAULT CURRENT_TIMESTAMP,
      deleted       BOOLEAN DEFAULT FALSE
  );
  CREATE INDEX IF NOT EXISTS idx_submissions_country ON submissions(country_code) WHERE deleted = FALSE;
  PRAGMA user_version = 1;
  ```
- `Insert(req SubmitRequest) (*SubmitResponse, error)` -- INSERT INTO submissions, return populated SubmitResponse with id, country_flag (from countryCodeToFlag), homelab_emoji (from HomelabEmojis map), created_at
- `GetAll() ([]SubmitResponse, error)` -- SELECT all non-deleted submissions, ordered by created_at ASC
- `Close() error`

**Step 4: Create `server/profanity.go`**
- Import `github.com/TwiN/go-away`
- Create a `ProfanityChecker` struct wrapping `goaway.ProfanityDetector`
- `NewProfanityChecker() *ProfanityChecker` -- creates detector. Note: go-away does NOT have a `WithExactWord()` method. Use the default detector: `goaway.NewProfanityDetector()`
- `IsProfane(text string) bool` -- returns true if profanity detected
- Add a `falsePosNames` slice with common name false positives: "Dick", "Fanny", "Willy", "Shittu", "Cockburn", "Hancock" -- and pre-check against this list before calling go-away (if name matches a known false positive, return false/not profane)

**Step 5: Create `server/sse.go`**
SSE Hub/Broadcaster pattern:
- `SSEHub` struct with `sync.RWMutex` and `subscribers map[chan []byte]struct{}`
- `NewSSEHub() *SSEHub`
- `Subscribe() chan []byte` -- buffered channel (capacity 16)
- `Unsubscribe(ch chan []byte)` -- lock, delete, close channel
- `Broadcast(data []byte)` -- RLock, iterate subscribers, non-blocking send (drop if full)
- `HandleSSE() http.HandlerFunc` -- sets headers (Content-Type: text/event-stream, Cache-Control: no-cache, Connection: keep-alive, X-Accel-Buffering: no), subscribes, defers unsubscribe, selects on r.Context().Done() and channel reads. Format: `event: submission\ndata: %s\n\n`

**Step 6: Create `server/handler.go`**
- `Handler` struct holding `*Store`, `*SSEHub`, `*ProfanityChecker`, and `baseURL string` (for QR code)
- `NewHandler(store *Store, hub *SSEHub, checker *ProfanityChecker, baseURL string) *Handler`
- `HandleSubmit() http.HandlerFunc` -- POST /api/submissions:
  1. Decode JSON body into SubmitRequest
  2. Trim whitespace from Name
  3. Validate with model.Validate() -- return 400 with ErrorResponse on failure
  4. Check profanity with checker.IsProfane(name) -- return 422 with `{Error: "That name wasn't accepted. Try a different one.", Field: "name"}`
  5. Insert via store.Insert() -- return 500 on DB error
  6. Marshal SubmitResponse to JSON, broadcast via hub.Broadcast()
  7. Return 201 with SubmitResponse JSON
- `HandleQRCode() http.HandlerFunc` -- GET /api/qr:
  1. Generate QR code PNG for `h.baseURL` using `qrcode.Encode(h.baseURL, qrcode.Medium, 512)`
  2. Set Content-Type: image/png
  3. Write PNG bytes to response
- `HandleHealth() http.HandlerFunc` -- GET /api/health:
  1. Return 200 with `{"status":"ok"}`

**Step 7: Create `server/middleware.go`**
- `CORSMiddleware(next http.Handler) http.Handler` -- sets Access-Control-Allow-Origin: *, Access-Control-Allow-Methods: GET, POST, OPTIONS, Access-Control-Allow-Headers: Content-Type. Handles OPTIONS preflight with 204.
- `LoggingMiddleware(next http.Handler) http.Handler` -- logs method, path, status code, duration to stdout using `log.Printf`
- Use a `responseWriter` wrapper to capture status code for logging.

**Step 8: Create `main.go`**
- Parse flags/env: `-addr` (default `:8080`), `-db` (default `guestbook.db`), `-base-url` (default `http://localhost:8080`)
- Open store with NewStore(dbPath)
- Create SSEHub, ProfanityChecker, Handler
- Create `http.NewServeMux()` (NEVER use DefaultServeMux)
- Register routes with Go 1.22+ method patterns:
  - `POST /api/submissions` -> handler.HandleSubmit()
  - `GET /api/submissions/stream` -> hub.HandleSSE()
  - `GET /api/qr` -> handler.HandleQRCode()
  - `GET /api/health` -> handler.HandleHealth()
- Wrap mux with CORSMiddleware then LoggingMiddleware
- Serve with `http.ListenAndServe(addr, wrappedMux)`
- Log startup message: `log.Printf("Server listening on %s", addr)`
